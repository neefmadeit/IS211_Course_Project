# IS211_Course_Project

Book Catalogue Web App
This application allows users to:

Log in
Add books using ISBN (via Google Books API)
View saved books
Delete books

The system uses:

Flask for backend
SQLite for database
Google Books API for fetching book data

Database Model:

books(id, title, author, pages, rating)

Run:
pip install flask requests
python app.py
Login: admin / password
